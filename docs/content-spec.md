# コンテンツ仕様

## スタート画面

スタート画面では、題名「スーパーマリ漢＜OTOKO＞」を表示し、プレイヤーがゲームを開始できる導線を明確にします。

スタート画面には、マリ漢がここまでたどってきた足跡を3行で表示してください。足跡は物語文、記号、短い詩のような表現などで構いませんが、必ず3行に分けます。

例:

```text
土管の向こうで、漢は一歩目を刻んだ。
雲より高い足場に、二歩目の気合が残った。
そして今、三歩目がこのステージへ続いている。
```

## ステージ1

ステージ1は、初期完成版で遊べる唯一のステージです。長い探索や複雑な分岐よりも、短時間で最後まで到達できることを優先します。

ステージには、足場、穴、敵、パワーアップ、ゴールを含めることを推奨します。配置はクラシックな横スクロールアクションの読みやすさを参考にしつつ、既存作品のステージ配置を複製しないでください。

## 敵

敵は、踏んで倒せる基本敵を少なくとも1種類用意することを推奨します。

初期完成版では敵の種類を増やしすぎず、次の役割が伝われば十分です。

- 踏むと倒せる。
- 横から触れると危険。
- 漢ボールで倒せる、または耐久値を減らせる。

## アイテムとパワーアップ

アイテムは、プレイヤーにパワーアップや得点、回復などの効果を与える要素として扱います。

初期完成版では、漢ボールを撃てる状態になるパワーアップを優先します。アイテムの見た目や名前はオリジナルにし、既存作品のアイテムを直接模倣しないでください。

## クリア演出

ステージ終端またはゴールへ到達したら、クリア演出へ遷移します。

クリア演出では、プレイヤーがステージを突破したことを明確に伝えます。必要に応じて、スタート画面へ戻る操作や最初から遊び直す操作を用意します。

## UI文言

UI文言、コメント、ドキュメント、プルリクエスト本文、レビュー返信は基本的に日本語で書きます。ゲーム内の表示も日本語を基本にし、プレイヤーが次に何をすればよいかを簡潔に示します。
