# 動作確認

## 手動確認チェックリスト

初期完成版の実装時は、少なくとも次を確認します。

- ブラウザで `index.html` を直接開いて起動できる。
- オフライン状態でも起動、プレイ、クリアができる。
- スタート画面、ステージ1、クリア演出の順に遷移する。
- 残機が尽きたとき、スタート画面へ戻る。
- スタート画面に題名「スーパーマリ漢＜OTOKO＞」が表示される。
- スタート画面にマリ漢がたどってきた足跡が3行で表示される。
- プレイヤーが左右移動、ジャンプ、敵踏み、パワーアップをできる。
- 漢ボールを撃てる。
- 漢ボールが画面内に最大2つまでしか存在しない。
- 画面内に漢ボールが2つある状態で3つ目の発射入力をしても、3つ目が生成されない。
- ステージ終端またはゴール到達後にクリア演出へ進む。
- UI文言やコメントが基本的に日本語になっている。

## 操作確認

初期版では、次の操作を確認します。

- 左右矢印キー、A、Dでプレイヤーが左右移動する。
- Space、W、上矢印キーでプレイヤーがジャンプする。
- Z、Jで漢ボールを撃つ。
- Enterでスタート画面から開始する。
- クリア演出またはゲームオーバー後、指定された操作でスタート画面へ戻る。

## ドキュメント確認

仕様や方針を変更した場合は、次も確認します。

- `README.md` から `AGENTS.md` と `docs/README.md` に辿れる。
- `AGENTS.md` が厚くなりすぎず、詳細仕様が `docs/` 側に整理されている。
- 既存の仕様が欠落なく、いずれかのdocsに整理されている。
- 実装者が `README.md`、`docs/README.md`、個別仕様の順で迷わず読める。

## 初期完成版の受け入れ条件

- ローカルのHTMLファイルからゲームを開始できる。
- スタート画面に題名と足跡3行が表示される。
- 短いステージ1をプレイできる。
- プレイヤーが走る、跳ぶ、敵を踏む、パワーアップする。
- 漢ボールを撃てる。
- 漢ボールの同時存在数が最大2つに制限されている。
- ステージを最後まで進むとクリア演出が表示される。
- ゲームオーバー時にスタート画面へ戻る。
