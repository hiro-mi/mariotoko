# 0001: 最小プレイアブル版

## 目的

`index.html` を直接ブラウザで開いて、スタート画面、ステージ1、クリア演出まで遊べる最小プレイアブル版を作る。

## 実装内容

- Canvas 2Dでゲーム画面を描画する。
- `start`、`play`、`clear`、`gameover` の画面状態を用意する。
- スタート画面に題名と足跡3行を表示する。
- Enterでステージ1を開始する。
- 左右移動とジャンプを実装する。
- 矩形の足場、穴、敵1種類、ゴールを配置する。
- 敵は上から踏むと倒れ、横から触れるとミスにする。
- 残機は3から開始し、尽きたらゲームオーバーへ遷移する。
- ゴール到達でクリア演出へ遷移する。

## 対象ファイル

- `index.html`
- `css/style.css`
- `js/main.js`
- `README.md`
- `docs/plans/completed/0001-minimum-playable.md`

## 確認項目

- `index.html` をブラウザで直接開いて起動できる。
- スタート画面に題名と足跡3行が表示される。
- Enterでステージ1へ遷移する。
- 左右矢印キー、A、Dで移動できる。
- Space、W、上矢印キーでジャンプできる。
- 敵を上から踏むと倒せる。
- 敵に横から触れる、または穴に落ちると残機が減る。
- 残機が0になるとゲームオーバーになり、スタートへ戻れる。
- ゴール到達でクリア演出へ進む。
- オフラインで動作し、外部ネットワーク依存がない。
