# ゲームデザイン

## コンセプト

「スーパーマリ漢＜OTOKO＞」は、ブラウザで動作する短編2D横スクロールアクションゲームです。短い1ステージを最後まで遊べる小さな完成版を優先します。

クラシックな横スクロールジャンプアクションの文法を参考にしつつ、既存作品そのものの再現ではなく、オリジナル作品として実装します。画像、音楽、ロゴ、キャラクター、ステージ配置、固有設定は独自のものにします。

最大の独自性は、ファイヤーボールの代わりに「漢ボール」を撃つことです。漢ボールは画面内に最大2つまで存在でき、攻撃のテンポや画面上の弾数管理が遊びの手触りになります。

## 画面遷移

基本の流れは次の通りです。

1. スタート画面
2. ステージ1
3. クリア演出

残機をすべて失ってゲームオーバーになった場合は、スタート画面へ戻します。

## 初期完成版の範囲

初期完成版では、見た目や音が最小限でも最後まで遊べることを優先します。

- スタート画面に題名と足跡3行を表示できる。
- 短いステージ1をプレイできる。
- プレイヤーが左右移動、ジャンプ、敵踏み、パワーアップをできる。
- 漢ボールを撃てる。
- 漢ボールの同時存在数を最大2つに制限できる。
- ステージ終端またはゴール到達でクリア演出へ進む。
- ゲームオーバー時にスタート画面へ戻る。

## 初期スコープに含めないもの

- 複数ステージ
- 複数ボス
- ステージ選択
- 成長要素
- 装備要素
- セーブデータ
- 長い探索や複雑な分岐
- オンライン接続を前提にした機能

## 完成の定義

`index.html` をブラウザで直接開いてゲームを開始でき、スタート画面、ステージ1、クリア演出まで一通り遊べる状態を初期完成版とします。
