# ゲームプレイルール

## 入力

入力はPCキーボードを前提にします。初期版のキー割り当ては次の通りです。

- 左右移動: 左右矢印キー、A、D
- ジャンプ: Space、W、上矢印キー
- 漢ボール: Z、J
- 開始または決定: Enter

## 移動とジャンプ

プレイヤーは左右へ走り、地面や足場の上でジャンプできます。ジャンプの高さ、加速度、最高速度などの数値は実装時に調整して構いませんが、短いステージを気持ちよく進めることを優先します。

初期完成版では、複雑なアクションよりも基本操作の読みやすさを優先します。二段ジャンプ、ダッシュ、壁キックなどは初期スコープに含めません。

## 敵踏み

敵は上から踏むと倒せることを基本にします。横や下から接触した場合はプレイヤーが被弾します。

踏み判定は厳しすぎない矩形判定を基本にし、短編ゲームとして最後まで遊びやすい調整を優先します。

## パワーアップ

パワーアップは、プレイヤーの状態をわかりやすく変化させる要素として扱います。

初期完成版では、少なくとも漢ボールを撃てる状態を表現できれば十分です。パワーアップの種類や段階は増やしすぎず、ステージ1を最後まで遊ぶために必要な範囲に留めます。

## 漢ボール

漢ボールは、プレイヤーが撃つ遠距離攻撃です。

- 漢ボールは画面内に最大2つまで存在できます。
- 画面内に漢ボールが2つ存在するとき、3つ目の発射入力は無視します。
- 漢ボールが敵に当たった場合は敵を倒すか、敵ごとの耐久値を減らします。
- 漢ボールが画面外へ出た場合、または地形や敵に当たって消えた場合は、同時存在数の枠を空けます。
- 漢ボールの見た目は、既存作品の弾を模倣せず、「漢」らしさが伝わる独自表現にします。

## 被弾と落下

敵、敵弾、危険地形、穴などに接触した場合は被弾またはミスとして扱います。

被弾や落下の詳細な数値、無敵時間、復帰位置は実装時に調整して構いません。ただし、次の方針を守ります。

- 被弾時はプレイヤー状態を1段階弱くするか、残機を失う。
- 穴へ落下した場合は残機を失う。
- 残機が残っている場合はステージの開始地点または安全な復帰地点から再開する。
- 残機がない場合はゲームオーバーとし、スタート画面へ戻る。

## 残機とゲームオーバー

残機はプレイヤー全体で管理します。残機をすべて失った場合はゲームオーバーとし、スタート画面へ戻します。

残機数の初期値は、短いステージを最後まで確認しやすい値にします。初期実装では3を推奨します。

## 当たり判定

当たり判定は初期完成版ではシンプルな矩形判定を基本にします。見た目より厳しすぎる判定は避け、短いゲームとして最後まで遊びやすい調整を優先します。
