# スーパーマリ漢＜OTOKO＞

「スーパーマリ漢＜OTOKO＞」は、ブラウザで動作するローカル2D横スクロールアクションゲームです。

クラシックな横スクロールジャンプアクションの文法を参考にしつつ、画像、音楽、効果音、ロゴ、キャラクター、ステージ配置、固有設定はオリジナルのものとして制作します。最大の特徴は、ファイヤーボールの代わりに「漢ボール」を撃てることです。漢ボールは画面内に最大2つまで存在できます。

## 現在の状態

最小プレイアブル版を実装済みです。`index.html` をブラウザで直接開くと、スタート画面、ステージ1、クリア演出まで一通り遊べます。

現在の版では、漢ボール、パワーアップ、音、画像アセットはまだ未実装です。

## 起動方針

ゲーム本体は、オンライン接続なしで動作するローカルHTMLゲームとして作ります。

- 初期実装はプレーンなHTML/CSS/JavaScriptを基本にします。
- `index.html` をブラウザで直接開いて遊べる状態を優先します。
- CDN、外部API、オンライン画像、オンライン音声、Webフォントには依存しません。
- 画像、音声、フォントなどのリソースが必要な場合は、すべてリポジトリ内のローカルファイルとして配置します。

## 遊び方

`index.html` をブラウザで直接開いてください。ビルドやローカルサーバーは不要です。

- 左右移動: 左右矢印キー、A、D
- ジャンプ: Space、W、上矢印キー
- 開始・決定: Enter

## 初期完成版の目標

- スタート画面に題名と、マリ漢がたどってきた足跡を3行で表示する。
- 短いステージ1をプレイできる。
- プレイヤーが走る、跳ぶ、敵を踏む、パワーアップする。
- 漢ボールを撃てる。
- 漢ボールが画面内に最大2つまでしか存在しない。
- ステージを最後まで進むとクリア演出へ遷移する。
- ゲームオーバー時にスタート画面へ戻る。
